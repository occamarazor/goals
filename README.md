## Runbook

### Server
```
cd src/server
```

#### Install project dependencies
```
yarn install
```

#### Set Environment Variables
```
mv .env.config .env
```

#### Lint
```
yarn lint
```

#### Prettify
```
yarn format
```

#### Run locally
```
yarn start
```
