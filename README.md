## Runbook

### Install project dependencies
```
yarn -i
```

### Server
#### Change directory to server
```
cd server
```

#### Install server dependencies
```
yarn -i
```

#### Set server environment variables
```
mv .env.config .env
```

#### Lint & format server files
```
yarn lint
```

#### Run server locally
```
yarn start
```
